# V.O.I.D. — Visual Overview of Income & Debt

Личный финансовый трекер. Отслеживай расходы по категориям с месячными лимитами, веди учёт активов и кредитов, рассчитывай рост накоплений. Данные хранятся локально на устройстве и синхронизируются с Google Sheets по желанию.

---

## Возможности

**Бюджет**
- Категории расходов с месячными лимитами и прогресс-барами
- Группировка категорий по цвету — одинаковый цвет = одна группа, суммарная статистика по группе
- Быстрое добавление расхода по тапу на категорию
- Навигация по месяцам, выбор месяца из пикера

**День**
- Список расходов за выбранный день
- Добавление и редактирование записей
- Переключение дней стрелками или выбор даты из календаря
- При повторном добавлении в ту же категорию — сумма добавляется (или перезаписывается — выбираешь галочкой)

**Аналитика**
- Расходы по месяцам (последние 6)
- Структура расходов текущего месяца — горизонтальные бары с процентами
- Доходы vs Расходы по месяцам
- Расходы по группам — stacked bar, видно как менялась структура трат

**Активы**
- Учёт банковских счетов и кредитных карт (кредиты вычитаются из итога)
- График роста накоплений
- История записей с возможностью редактирования и удаления по дате

**Доходы**
- Источники дохода по месяцам
- Баланс: доходы минус расходы

**Калькулятор вкладов**
- Расчёт роста депозита с процентами и ежемесячным пополнением
- Таблицы «По годам» и «По месяцам»
- Стартовая сумма подтягивается из текущих активов

**Настройки**
- Управление категориями: добавление, переименование, цвет, группировка
- Управление банками и кредитными счетами
- Синхронизация с Google Sheets
- Экспорт данных в CSV
- Очистка всех данных

## Установка PWA

PWA устанавливается прямо из браузера — никакого App Store не нужно.

### iPhone / iPad (Safari)
1. Открой приложение в **Safari**
2. Нажми кнопку **Поделиться** (квадрат со стрелкой вверх)
3. Выбери **«На экран «Домой»»**
4. Нажми **Добавить**

Приложение появится на главном экране как обычное, работает в полноэкранном режиме.

### Android (Chrome)
1. Открой приложение в **Chrome**
2. Нажми **⋮** (три точки) → **«Добавить на главный экран»**
3. Нажми **Добавить**

Или Chrome сам предложит установку — появится баннер внизу экрана.

### Компьютер (Chrome / Edge)
1. В адресной строке нажми иконку **установки** (📥) справа
2. Нажми **Установить**

## Синхронизация с Google Sheets

Синхронизация необязательна — приложение работает полностью офлайн. Если хочешь резервную копию в таблице и доступ с нескольких устройств:

### Шаг 1 — Создать таблицу

1. Открой [Google Sheets](https://sheets.google.com)
2. Создай **новую пустую таблицу**
3. Назови её как угодно, например «V.O.I.D. бюджет»

### Шаг 2 — Добавить Apps Script

1. В таблице открой **Расширения → Apps Script**
2. Удали весь код в редакторе (там будет пустая функция)
3. В приложении зайди в **Настройки → Скопировать Apps Script**
4. Вставь скопированный код в редактор Apps Script
5. Нажми **💾 Сохранить** (Ctrl+S)

### Шаг 3 — Задеплоить

1. Нажми **Развернуть → Новое развёртывание**
2. Тип: **Веб-приложение**
3. Параметры:
   - Выполнять как: **Я**
   - Кому разрешён доступ: **Все**
4. Нажми **Развернуть**
5. Подтверди доступ (Google покажет предупреждение — нажми «Дополнительно» → «Перейти»)
6. Скопируй **URL веб-приложения**

### Шаг 4 — Подключить в приложении

1. Зайди в **Настройки → URL веб-приложения**
2. Вставь скопированный URL
3. Нажми **Сохранить**
4. Нажми **⚡ Проверить подключение** — должно появиться «✓ Подключено»

После подключения:
- Приложение автоматически синхронизируется каждые 15 секунд (если были изменения)
- При первом открытии загружает данные из таблицы
- Вручную: **Загрузить из таблицы** или **Выгрузить в таблицу**

### Что создаётся в таблице

При первой синхронизации автоматически создаются листы:

| Лист | Описание |
|---|---|
| `По дням 2026` | Матрица расходов: строки = категории, столбцы = даты |
| `Шаблон` | Шаблон для месячных листов с лимитами |
| `Январь 2026` и т.д. | Месячные листы с лимитами по категориям |
| `Активы` | История активов по банкам |
| `Доходы` | Источники дохода (скрытый) |
| `Комментарии` | Комментарии к расходам (скрытый) |
| `Настройки` | Цвета категорий (скрытый) |

Каждый год добавляется новый лист `По дням YYYY` — данные за разные годы хранятся раздельно.

### Обновление скрипта

Если вышла новая версия приложения:
1. Скопируй новый код через **Настройки → Скопировать Apps Script**
2. Вставь в редактор Apps Script
3. **Развернуть → Управление развёртываниями → ✏️ Редактировать**
4. Версия: **Новая версия** → Развернуть

## Структура проекта

```
void-app/
├── index.html          ← HTML + весь JS (собирается из js/)
├── manifest.json       ← PWA манифест
├── css/app.css         ← Стили
├── js/
│   ├── db.js           ← База данных, хелперы, toast, модалки
│   ├── nav.js          ← Навигация, переключение месяцев
│   ├── budget.js       ← Бюджет, расходы, лимиты
│   ├── day.js          ← Вкладка День
│   ├── income.js       ← Доходы
│   ├── stats.js        ← Аналитика, графики
│   ├── assets.js       ← Активы, банки
│   ├── settings.js     ← Настройки, категории
│   ├── sync.js         ← Синхронизация Google Sheets
│   ├── calc.js         ← Калькулятор вкладов
│   └── init.js         ← Инициализация, buildPayload, mergePullData
└── apps-script/
    └── Code.gs         ← Google Apps Script
```
